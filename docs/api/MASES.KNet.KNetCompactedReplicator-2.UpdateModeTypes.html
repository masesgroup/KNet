Fields

Name Description
OnConsume

The KNetCompactedReplicator<TKey, TValue> is updated only after an update is consumed from Kafka, even if the add or update is made locally by the current instance
OnConsumeSync

The KNetCompactedReplicator<TKey, TValue> is updated only after an update is consumed from Kafka, even if the add or update is made locally by the current instance. Plus the update waits the consume of the data before unlock
OnDelivery

The KNetCompactedReplicator<TKey, TValue> is updated as soon as an update is delivered to Kafka by the current application
