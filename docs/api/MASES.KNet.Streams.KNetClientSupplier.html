Remarks

The default KafkaClientSupplier used from KafkaStreams allocates both KafkaConsumer and KafkaProducer. KNetClientSupplier uses KNetConsumer<K, V> and KNetProducer<K, V>, the instance shall be disposed when no more in use, but still be alive still KafkaStreams is alive.
